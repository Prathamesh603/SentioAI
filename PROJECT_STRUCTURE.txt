🚀 HOW TO USE
================================================================================

1. SETUP (5 minutes):
   ───────────────────────────────────────────────────────
   cd sentiment_analysis_chatbot
   python -m venv venv
   venv\Scripts\activate              (Windows)
   source venv/bin/activate           (macOS/Linux)
   pip install -r requirements.txt
   cp .env.example .env
   # Edit .env and add your Groq API key

2. RUN THE APPLICATION:
   ───────────────────────────────────────────────────────
   python run.py

3. ACCESS DASHBOARD:
   ───────────────────────────────────────────────────────
   Opens automatically at: http://localhost:8501

🐛 COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "GROQ_API_KEY not found"
→ Make sure .env file exists and contains your API key

Issue: "Port 8501 already in use"
→ streamlit run dashboard/main.py --server.port=8502

Issue: "ImportError: No module named 'streamlit'"
→ pip install -r requirements.txt

Issue: Scraper returns no data
→ Some websites block scrapers. Try different URLs.

💡 PRO TIPS
================================================================================

   • Lower temperature (0.3-0.5) for consistent responses
   • Keep context turns at 4-5 for speed
   • Clear history periodically

✨ For Better Scraping:
   • Use complete product URLs
   • Try e-commerce sites first
   • Include product name in your message
